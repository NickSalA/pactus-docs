# ContractIA Documentation Site

Sitio de documentacion generado con Astro + Starlight.

## Estructura

- `src/content/docs/`: paginas Markdown/MDX.
- `src/assets/`: imagenes y recursos.
- `public/`: archivos estaticos.

## Comandos

Ejecuta desde esta carpeta:

```bash
npm install
npm run dev
npm run build
npm run preview
```
